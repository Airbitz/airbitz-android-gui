Import Wallet Info


This screen will import a Bitcoin private key and sweep the funds into the currently selected wallet. Choose the destination wallet by tapping the top header button and selecting a wallet.

Private key import can be accomplished in several methods:

  • Scan a private key QR code using the camera
  • Tap [Address] to manually type or paste a private key into the text field
  • Tap the [Photos] button to choose a photo containing a private key QR code
